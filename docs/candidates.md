# コンテンツ候補

Day 13 まで HTML → CSS → JS の実行環境と進んできた。ここから Next.js までの道筋を整理する。

## JavaScript の言語

JS の文法を教えるのではなく、「なぜこうなっているか」を仕組みから語る。

| テーマ | 切り口 | ステータス |
|-------|--------|----------|
| Promise と非同期処理 | コールバック → Promise → async/await の変遷。イベントループにも触れる | 有力 |
| 関数の変遷と使い分け | function → アロー関数。`this` の束縛問題 | 有力 |
| 動的型付け | JS は型がなくても動く。なぜそうなっているか、何が起きるか | 有力 |

## JavaScript のエコシステム

JS を取り巻くツールや仕組み。読者が触っているのに意味がわかっていないもの。

| テーマ | 切り口 | ステータス |
|-------|--------|----------|
| npm | 3 つの意味（CLI / レジストリ / 企業）、パッケージ、package.json、node_modules | 作成中（drafts/001） |

## React / UI ライブラリ

なぜ React が必要になったか。
