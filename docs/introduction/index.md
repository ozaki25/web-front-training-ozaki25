# はじめに

## このコンテンツについて

HTML/CSS の基礎から Next.js まで、Web フロントエンドの知識の引き出しを増やしていくコンテンツです。

Web フロントエンドには多くの技術や概念があります。すべてを深く理解する必要はありませんが、「聞いたことがある」という引き出しがあるだけで大きな差が生まれます。

- AI に指示を出すとき、的確な言葉で伝えられる
- AI が返してきたコードに違和感があったとき、何がおかしいか気づける
- チームの技術的な会話で、話題についていける

このコンテンツは、そういった引き出しを毎日少しずつ増やしていくためのものです。

::: tip このコンテンツの作り方
文章の執筆には AI を活用しています。ただし、何を伝えるか・どんな順番で伝えるかの設計と、内容の大枠のチェックは人間が行っています。AI 任せの自動生成ではなく、作者の意図を持って作っているコンテンツです。
:::

## 進め方

- 1日1レッスン、15分程度で読めます
- 順番に進めてください（前のレッスンの知識を前提にしています）
- 暗記する必要はありません。「こういう概念があるんだな」くらいの感覚で大丈夫です
- コンテンツは当日の朝に公開されます。それまでお待ちください
