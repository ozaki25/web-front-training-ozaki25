# Day 1: HTML の基本構造

## 今日のゴール

- HTML ファイルの基本構造を理解する
- 見出し・段落・リストを使ってページを作れるようになる

## HTML とは

HTML（HyperText Markup Language）は、Web ページの**構造**を記述するための言語です。ブラウザはHTMLを読み取り、画面に表示します。

## 基本構造

すべての HTML ファイルは以下の構造を持ちます。

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>ページタイトル</title>
  </head>
  <body>
    <!-- ここに表示する内容を書く -->
  </body>
</html>
```

| 要素 | 役割 |
|------|------|
| `<!DOCTYPE html>` | HTML5 であることをブラウザに伝える |
| `<html>` | HTML 文書のルート要素 |
| `<head>` | メタ情報（タイトル、文字コードなど） |
| `<body>` | 画面に表示される内容 |

## よく使うタグ

### 見出し

`<h1>` が最も大きく、`<h6>` が最も小さい見出しです。

```html
<h1>大見出し</h1>
<h2>中見出し</h2>
<h3>小見出し</h3>
```

> **ポイント**: `<h1>` は1ページに1つが基本です。見出しレベルは飛ばさず順番に使いましょう。

### 段落

```html
<p>これは段落です。文章のまとまりを表します。</p>
<p>別の段落はpタグで分けます。</p>
```

### リスト

**順序なしリスト**（箇条書き）:

```html
<ul>
  <li>りんご</li>
  <li>みかん</li>
  <li>バナナ</li>
</ul>
```

**順序ありリスト**（番号付き）:

```html
<ol>
  <li>まず卵を割る</li>
  <li>かき混ぜる</li>
  <li>フライパンで焼く</li>
</ol>
```

## 実際に書いてみよう

ここまで学んだタグを組み合わせて、自己紹介ページを作ってみましょう。以下のコードを `index.html` というファイル名で保存してください。

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>自己紹介</title>
  </head>
  <body>
    <h1>山田太郎</h1>
    <p>東京都在住のエンジニアです。</p>
    <p>Web開発に興味があり、日々勉強しています。</p>

    <h2>趣味</h2>
    <ul>
      <li>読書</li>
      <li>ランニング</li>
      <li>プログラミング</li>
    </ul>
  </body>
</html>
```

保存したら、ファイルをブラウザで開いてみましょう。ダブルクリックするか、ブラウザにドラッグ&ドロップすれば表示されます。

名前や趣味の部分を自分の情報に書き換えて、もう一度ブラウザを更新してみてください。変更が反映されるはずです。

## まとめ

- HTML は `<!DOCTYPE html>` → `<html>` → `<head>` + `<body>` という構造
- `<h1>`〜`<h6>` で見出し、`<p>` で段落、`<ul>`/`<ol>` + `<li>` でリスト
- タグにはそれぞれ意味（セマンティクス）がある。見た目ではなく「何を表すか」で選ぶ
